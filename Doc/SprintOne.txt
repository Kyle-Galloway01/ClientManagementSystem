Seperating Docs by Sprints 
Sprint 1 -  Set up the development environment
Create a New Project:

Open Visual Studio Code.
Navigate to terminal and to directory to create project
Run the following command to create a new ASP.NET Core MVC project:
    dotnet new mvc -n ClientManagementSystem
This command will create a new ASP.NET Core MVC project named "ClientManagementSystem" in the current directory.
